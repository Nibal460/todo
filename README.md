# Todo Application

A Todo Management Application built with the Django Python Framework.

## Overview

This project is a Todo Application developed using Django and Python. It allows users to manage daily tasks efficiently through a simple and organized interface.

The application demonstrates core web development concepts such as task creation, updating, deletion, and database management using Django.

## Features

* Create new tasks
* Update existing tasks
* Delete tasks
* Mark tasks as completed
* User-friendly interface
* Database integration
* Django Admin Panel
* Task management system

## Technologies Used

* Python
* Django
* HTML
* CSS
* Bootstrap
* SQLite
* JavaScript

## Installation

### Clone the Repository

```bash
git clone https://github.com/Nibal460/todo.git
cd todo
```

### Create Virtual Environment

```bash
python -m venv env
```

### Activate Virtual Environment

#### Windows

```bash
env\Scripts\activate
```

#### Linux / macOS

```bash
source env/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Migrations

```bash
python manage.py migrate
```

### Create Superuser

```bash
python manage.py createsuperuser
```

### Start Development Server

```bash
python manage.py runserver
```

Open your browser:

```text
http://127.0.0.1:8000/
```
